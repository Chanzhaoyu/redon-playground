# playground

> A playground for testing out ideas.

## Start

```sh
# pnpm
npm install pnpm

# install
pnpm install

# start
pnpm start
```

## Run

Copy the .env.example file to .env and fill in the values.

```yaml
OPENAI_API_KEY=xxx
```

```sh
# run dev
pnpm dev
```
